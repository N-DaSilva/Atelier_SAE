## BUT DU JEU :

Une conversation a été trouvée qui pourrait bien vous aider à trouver la coupable derrière l'usurpation d'identité d'Alice ! Malheureusement celle-ci est chiffrée... Armés de vos fiches de décryptage, à vous de trouver sur quels boutons appuyer pour décoder les messages et avoir accès à la conversation !

## MIS EN PLACE :

Le jeu de décryptage est lancé une fois le tableau de suivi d'enquête complété.

Pour jouer avec les boutons :
1) Installer le logiciel Chataigne
2) Ouvrir le fichier controllers.noisette
3) Remapper si nécessaire pour correspondre à l'input choisi (chaque bouton doit être mappé aux touches du pavé numérique de 0 à 9, où 0 correspond à 10)

## COMMENT JOUER :

1) Choisir la difficulté (et mettez à disposition la ou les fiche(s) de décryptage correspondante(s)) et cliquez sur commencer.
2) Sur la page de terminal, appuyer sur n'importe quelle touche pour écrire les commandes. Appuyer sur Entrée passe à la ligne suivante.
3) Utiliser la page de calibrage pour vérifier la connexion des boutons, s'ils sont utilisés.
4) Lancer le jeu !
